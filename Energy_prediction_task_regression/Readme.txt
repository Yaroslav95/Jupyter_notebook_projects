Задача: Предсказать энергопотребление цеха на сутки вперёд.
----------------------------------------------------------------------------------------------------------
Для запуска необходимо:
1) Установить библиотеки, представленные в requirements.txt. Если такие уже установлены, то обратить внимание
   на версии, при не совпадении могут быть ошибки.
2) Запустить файл LightGBM_model.ipynb
----------------------------------------------------------------------------------------------------------
Описание файлов проекта:
В файлах requirements представлены версии используемых библиотек: requirements.txt содержит все необходимые 
зависимости, а в requirements_full.txt представлена полная выгрузка (с учетом установленного виртуального
окружения).

Проект (папка source_code) состоит из следующих элементов:
1) LightGBM_model.ipynb - main файл, можно полностью запускать, время отработки примерно 35 минут.
   Время работы обусловлено тем, что алгоритм проходит 847 итераций (обучение каждые сутки и прогноз 
   на следующие). Для новых суток потребуется только одна итерация.
2) Папка data содержит файлы с исходными данными.
3) Папка const содержит файл objects_file.py, с различными объектами, которые вынесены в отдельный файл.
4) Папка functions включает различные функции.
5) Папка other_files содержит 2 папки: hyperopt(Hyperopt_params.ipynb) - здесь представлена оптимизация
   параметров LightGBM, по результатам которой выбран наилучший вариант.
   sarimax(Sarimax_model.ipynb) - работа с временным рядом, не вошедшая в окончательный проект.
